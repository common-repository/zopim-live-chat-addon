== Installation ==
1. Upload the `zopim-live-chat-addon` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions == 

Install the plugin, in the Zopim menu options you will have another menu item named "Zopim Pages Exclusion".
Add there Pages ID's separated by commas.
